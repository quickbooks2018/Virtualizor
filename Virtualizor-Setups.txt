###Installation###
For RHEL & CentOS 7+

firewall-cmd --zone=public --permanent --add-port=4081-4085/tcp
firewall-cmd --zone=public --permanent --add-port=5900-6000/tcp 

wget -N http://files.virtualizor.com/install.sh 
chmod 0755 install.sh 
./install.sh email=your-email kernel=kvm lvg=virtualizor


cat /proc/mdstat

umount /dev/md1

fdisk -l

fdisk /dev/md124

RAID AND LVM COMBINATION ON ONLY WORKS AT STARTUP INSTALLATION 

(if you created via command line, it is not persistent, in such go for file base storage with RAID)

shutdown -r now

after reboot

pvdisplay

###Now make sure remove an entry from in /etc/fstab ----> /dev/md1 UUID=123225 /data ###
